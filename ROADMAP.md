# Планы развития

## v2.1 — Новые стили и форматы

- Новые стили логотипов: `holographic`, `retro`, `minimal`, `glass`
- Новые стили иконок: `shield`, `bolt`, `gem`, `moon`
- Экспорт в SVG
- Поддержка экспорта WebP

## v2.2 — Движок компоновки

- Многострочный текст для логотипов
- Комбинированные макеты «иконка + текст» (logo lockup)
- Настраиваемые отступы и выравнивание
- Пресеты соотношений сторон (квадрат, широкий 16:9, баннер 3:1)

## v2.3 — Темы

- Пресеты тем («корпоративная», «neon-city», «luxury», «minimal-dark»)
- JSON-файлы тем — пользователь задаёт палитры без изменения кода
- Извлечение цветовой палитры из изображения (доминантный цвет как основной)

## v2.4 — Web API

- HTTP-сервер на Express/Fastify, оборачивающий `generateLogo` и `generateIcon`
- REST-эндпоинты: `POST /logo`, `POST /icon`, `GET /styles`
- Docker-образ для self-hosted деплоя

## v3.0 — Архитектура плагинов

- Система плагинов для кастомных рендереров
- Отдельные npm-пакеты для каждого семейства стилей
- Браузерная сборка через `OffscreenCanvas`
